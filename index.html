O que é a tabela periódica?

A tabela periódica é uma tabela que organiza os elementos químicos com base nas suas propriedades.

Quem foi o autor de 'Dom Casmurro'?

O que é a fotossíntese?

A fotossíntese é o processo pelo qual as plantas transformam luz solar em energia.
